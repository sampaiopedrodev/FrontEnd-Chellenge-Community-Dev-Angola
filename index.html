COOL.CSS

Uma nova esperiência para desenvolvimento web front-end

Como Usar?

Para usar o COOL.CSS baixar o arquivo COOL.CSS.rar, extrair os arquivos para qualquer directorio e importar o arquivo cool.css, e pronto, pode começar a desenvolver.

<link rel="stylesheet" href="./css/cool.css">
